# SHARPER GYM WEBSITE

A modern and fully responsive gym website developed using **HTML, CSS, and JavaScript**. The project is designed to provide an engaging user experience while showcasing gym services, workout programs, trainer profiles, membership plans, and contact information.

## FEATURES

* Responsive design for desktop, tablet, and mobile devices
* Modern and attractive user interface
* Interactive navigation menu
* About Us section
* Workout Programs showcase
* Trainer profiles section
* Membership plans display
* Contact information section
* User registration and feedback form
* Mobile-friendly navigation with hamburger menu

## TECHNOLOGIES USED

* HTML
* CSS
* JavaScript
